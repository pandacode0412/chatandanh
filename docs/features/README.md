# Feature Specs

Mỗi tính năng lớn nên có một file spec riêng trong thư mục này.

Quy ước tên file:

```text
<phase>-<feature-slug>.md
```

Ví dụ:

```text
phase-2-save-connections.md
phase-4-video-call.md
```

Tạo feature spec mới bằng template:

- [Feature Spec Template](../templates/feature-spec-template.md)

Không implement tính năng lớn khi chưa có:

- User flow.
- Vietnamese UI copy.
- Acceptance criteria.
- API/socket impact.
- Data/privacy impact.
- Test plan.
