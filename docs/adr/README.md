# Architecture Decision Records

ADR dùng để ghi lại các quyết định kỹ thuật quan trọng.

Tạo ADR mới khi thay đổi:

- Auth/session.
- Data model hoặc retention.
- Matching algorithm.
- Privacy/anonymity model.
- Moderation/safety.
- Realtime protocol.
- Infrastructure.
- Audio/video call architecture.

Quy ước tên file:

```text
YYYY-MM-DD-short-title.md
```

Tạo ADR bằng template:

- [ADR Template](../templates/adr-template.md)
